## How to 

- clone repository 

Dependencies :
- ```git``` 
- ```flask```
- ```node```

Running the Web App:
- ```$ npm install```
- ```$ npm run dev```
this will deploy the web page
then, open a second terminal and run 
- ```$ python api/index.py```
this will start the flask server so that the pipeline will function
